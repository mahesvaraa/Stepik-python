**–§–æ—Ä–º–∞—Ç –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**

–ü—Ä–æ–≥—Ä–∞–º–º–∞ –¥–æ–ª–∂–Ω–∞ –≤—ã–≤–µ—Å—Ç–∏ –æ–¥–Ω–æ —á–∏—Å–ª–æ ‚Äî –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –≤ –∑–∞—à—Ç—Ä–∏—Ö–æ–≤–∞–Ω–Ω–æ–π –æ–±–ª–∞—Å—Ç–∏ –∫–≤–∞–¥—Ä–∞—Ç–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã.

**Sample Input 1:**

```
3
1 4 5
6 7 8
1 1 6
```

**Sample Output 1:**

```
8
```

**Sample Input 2:**

```
4
0 1 4 6
0 0 2 5
0 0 0 7
0 0 0 0
```

**Sample Output 2:**

```
7
```

**Sample Input 3:**

```
2
6 0
7 9
```

**Sample Output 3:**

```
9
```

# Solution

```python
n = int(input())

lst = [input().split() for i in range(n)]
a = []
for i in lst:
    lst[lst.index(i)] = [int(j) for j in i]
for i in range(n):
    for j in range(n):
        if (j <= i <= n - 1 - j) or (j >= i >= n - 1 - j):
            a.append(lst[i][j])
print(max(a))
```
