# 10.2 Битовые операции И, ИЛИ, НЕ, XOR

Подвиг 5. На вход программы подается целое десятичное число. Используя битовые операции, выполните умножение введенного
числа на 4. Результат отобразите на экране.

**Sample Input:**

```
40
```

**Sample Output:**

```
160
```

# Solution

```python
print(int(input()) << 2)
```
