**–§–æ—Ä–º–∞—Ç –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**

–ü—Ä–æ–≥—Ä–∞–º–º–∞ –¥–æ–ª–∂–Ω–∞ –≤—ã–≤–µ—Å—Ç–∏ —Ç–µ–∫—Å—Ç –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–∏ —Å —É—Å–ª–æ–≤–∏–µ–º –∑–∞–¥–∞—á–∏.

**Sample Input:**
```
5
abcdef
bcdefg
cdefgh
defghi
efghij
2
```
**Sample Output:**

bcdef

# Solution
```python
n = int(input())
a = [input() for i in range(n)]
k = int(input())
b = ""
for i in range(n):
    b += (a[i][k - 1:k])
print(b)
```
