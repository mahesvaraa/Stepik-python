# Solution
```python
s = input().split()
a = [s[0]]
count = 1
for i in s[1:]:
    if i == a[-1]:
        count += 1
    else:
        a[-1] = [a[-1]] * count
        count = 1
        a.append(i)
a[-1] = [a[-1]] * count
print(a)
```
