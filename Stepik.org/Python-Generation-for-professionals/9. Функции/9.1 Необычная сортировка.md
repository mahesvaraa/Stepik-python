# Solution

```python

string = list(input())
num_list = sorted([i for i in sorted(string) if i.isdigit()], key =lambda x: int(x) % 2 == 0)
chars = sorted([i for i in sorted(string) if i.isalpha()], key=lambda x: -x.islower())

print(''.join(chars + num_list))
```
