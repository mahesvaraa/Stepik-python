# Solution

```python
from fractions import Fraction as F

n = int(input())
minn = 0
for i in range(1, n):
    for j in range(1, n):
        x = F(i, j)
        if i + j == n and i < j and x.numerator + x.denominator == n:
            if F(i, j) > minn:
                minn = F(i, j)
print(minn)
```
