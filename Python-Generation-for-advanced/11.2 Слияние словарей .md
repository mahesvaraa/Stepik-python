–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ 2. –í—ã–∑—ã–≤–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏—é merge() –Ω–µ –Ω—É–∂–Ω–æ, —Ç—Ä–µ–±—É–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å. 

–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ 3. –°–ª–∏—è–Ω–∏–µ –ø—É—Å—Ç—ã—Ö —Å–ª–æ–≤–∞—Ä–µ–π –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –ø—É—Å—Ç—ã–º —Å–ª–æ–≤–∞—Ä–µ–º.

# Solution
```python
def merge(dicts):      # values - —ç—Ç–æ —Å–ø–∏—Å–æ–∫ —Å–ª–æ–≤–∞—Ä–µ–π
    
    super_dict = {}
    for d in dicts:
        for k, v in d.items():
            if super_dict.get(k) is None:
                super_dict[k] = []
            if v not in super_dict.get(k):
                super_dict[k].append(v)

    for k in super_dict.keys():
        super_dict[k] = set(super_dict[k])
    return super_dict
```
