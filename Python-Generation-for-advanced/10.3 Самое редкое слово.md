**Sample Input 1:**
```
home sweet home
```
**Sample Output 1:**
```
sweet
```
**Sample Input 2:**
```
home sweet home sweet.
```
**Sample Output 2:**
```
home
```
# Solution
```python
import re

s = re.sub(r"[.,!?:;-]", "", input()).lower().split()

result = {}
for i in s:
    result[i] = s.count(i)
print(sorted([i for i in result if result.get(i) == min(result.values())])[0])
```
