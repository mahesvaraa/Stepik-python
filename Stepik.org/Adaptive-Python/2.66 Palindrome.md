# Palindrome

Given a string. Find whether it is a palindrome, i.e. it reads the same both left-to-right and right-to-left. Output
“yes” if the string is a palindrome and “no” otherwise.

**Sample Input:**

```
kayak
```

**Sample Output:**

```
yes
```

# Solution

```
word = input().lower()
print(('no', 'yes')[word == word[::-1]])
```
