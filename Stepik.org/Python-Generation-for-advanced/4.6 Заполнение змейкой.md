# Solution

```python
n = input().split()
l = []
a, b = int(n[0]), int(n[1])
for i in range(1, a + 1):
    if i % 2 != 0:
        l.append([i for i in range(b * (i - 1) + 1, b * i + 1, 1)])
    else:
        l.append([i for i in range(b * i, b * (i - 1), -1)])
for i in range(a):
    for j in range(b):
        print(l[i][j], end=" ")
    print()
```
