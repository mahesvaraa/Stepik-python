# 2.1 Методы экземпляра. Аргумент self
Создайте класс Lion. В нем должен быть метод roar, который печатает на экран "Rrrrrrr!!!"

Пример работы с классом Lion
```python
simba = Lion()
simba.roar() # печатает Rrrrrrr!!!
```

# Solution
```python
class Lion():
    def roar(self):
        print('Rrrrrrr!!!')
```
