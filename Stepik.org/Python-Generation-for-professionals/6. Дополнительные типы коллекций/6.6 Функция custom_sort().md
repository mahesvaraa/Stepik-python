# Solution

```python
from collections import OrderedDict


def custom_sort(ordered_dict, by_values=False):
    for k, v in sorted(ordered_dict.items(), key=lambda x: x[1] if by_values else x[0]):
        ordered_dict[k] = ordered_dict.pop(k)
```
