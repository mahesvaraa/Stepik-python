**Sample Input 1:**

```

4 1 2 3
1 2 3 4
```

**Sample Output 1:**

```

```

**Sample Input 2:**

```
3
1 2 3
3 2 1
2 3 4
```

**Sample Output 2:**

```
NO
```

# Solution

```python
n = int(input())
a = [input().split() for i in range(n)]
x = list(zip(*a))
s = set(map(str, range(1, n + 1)))
flag, flag1 = False, False
for i in a:
    flag = set(i) == s
for i in x:
    flag1 = set(i) == s
print(["NO", "YES"][all([flag, flag1])])
```
