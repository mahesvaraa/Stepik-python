# Solution

```python
d = {
    1: ".,?!:",
    2: "ABC",
    3: "DEF",
    4: "GHI",
    5: "JKL",
    6: "MNO",
    7: "PQRS",

    0: " "
}

    for i in d:
        if j in d.get(i):
            print(str(i) * (d.get(i).index(j) + 1), end="")
```
