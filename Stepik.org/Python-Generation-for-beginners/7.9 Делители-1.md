**Sample Input 1:**

**Sample Input 2:**

# Solution

```python
a, b = int(input()), int(input())
max_sum = 0
for i in range(b, a, -1):
    summ = 0
    for j in range(1, b + 1):
        if i % j == 0 and i != 1:
            summ += j
            if summ > max_sum:
                max_sum = summ
                max_var = i
print(max_var, max_sum)
```
