# Frequency of number

Given the natural number 1 â‰¤ n â‰¤ 10 and the array of integers A[1â€¦n], not exceeding 10^9 by absolute value. Output 1, if the array A contains the number occurring strictly more than n/2 times, and output 0 otherwise.

**Sample Input:**
```
5
2 3 9 2 2
```
**Sample Output:**
```
1
```
# Solution
```
# itertools
from itertools import *

n = int(input())
flag = 0
arr = list(map(int, input().split()))
for key, group in groupby(sorted(arr)):
    if len(list(group)) > n / 2:
        flag = 1
print(flag)

# collections
from collections import Counter
n = int(input())
list_values = map(int, input().split())

print(int(max(Counter(list_values).values()) * 2 > int(n)))
```
