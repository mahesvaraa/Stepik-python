**–§–æ—Ä–º–∞—Ç –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**

–ù–∞ –≤—Ö–æ–¥ –ø—Ä–æ–≥—Ä–∞–º–º–µ –ø–æ–¥–∞—ë—Ç—Å—è –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–æ–µ —á–∏—Å–ª–æ nn ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å—Ç—Ä–æ–∫ –∏ —Å—Ç–æ–ª–±—Ü–æ–≤ –≤ –º–∞—Ç—Ä–∏—Ü–µ, –∑–∞—Ç–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –º–∞—Ç—Ä–∏—Ü—ã: nn —Å—Ç—Ä–æ–∫, –ø–æ nn —á–∏—Å–µ–ª –≤ –∫–∞–∂–¥–æ–π, —Ä–∞–∑–¥–µ–ª—ë–Ω–Ω—ã–µ –ø—Ä–æ–±–µ–ª–∞–º–∏.

**–§–æ—Ä–º–∞—Ç –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**

–ü—Ä–æ–≥—Ä–∞–º–º–∞ –¥–æ–ª–∂–Ω–∞ –≤—ã–≤–µ—Å—Ç–∏ —Å–ª–æ–≤–æ YES, –µ—Å–ª–∏ –º–∞—Ç—Ä–∏—Ü–∞ —è–≤–ª—è–µ—Ç—Å—è –º–∞–≥–∏—á–µ—Å–∫–∏–º –∫–≤–∞–¥—Ä–∞—Ç–æ–º, –∏ —Å–ª–æ–≤–æ NO –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ.

**Sample Input 1:**
```

```
**Sample Output 1:**
```

```
**Sample Input 2:**
```

```
**Sample Input 3:**
```
3
4 9 2
3 5 7
8 1 6
```
**Sample Output 3:**
```

```
# Solution

```python
n = int(input())
a, res = [], []
summ1, summ2 = 0, 0

for i in range(n):
    a.append(input().split())

for i in range(n):
    a[i] = list(map(int, a[i]))

for i in range(n):
    res.append(sum(a[i]))
    summ = 0
    for j in range(n):
        summ += a[i][j]
        if i == j:
            summ1 += a[i][j]
        if i + j + 1 == n:
            summ2 += a[i][j]
    res.append(summ)

res.append(summ1)
res.append(summ2)
b = [i for i in range(1, n ** 2 + 1)]
sumb = sum(b)
suma = sum([sum(i) for i in a])
flag = all(res[i] == res[i + 1] for i in range(len(res) - 1))

if flag and sumb == suma and a[0] != a[1]:
    print("YES")
else:
    print("NO")
```
